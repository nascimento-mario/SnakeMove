A webcam-controlled Snake game where you control the snake using finger gestures!

## Features
- Point your index finger to move the snake (up/down/left/right)
- Close your fist to stop the snake
- Accessible design with slower speed for all ages
- Real-time hand tracking using MediaPipe

## How to Play
1. Download `index.html`
2. Open it in a modern browser (Chrome, Firefox, or Edge)
3. Allow camera permissions
4. Click "Start Camera"
5. Click "Start Game"
6. Point your finger to control the snake!
